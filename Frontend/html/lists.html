Lists and Tables

HTML provides elements for organizing information into lists and tables, allowing developers to present data in a structured and readable format. Understanding how to use lists and tables is crucial for creating well-formatted and visually appealing web content.

Ordered Lists

Ordered lists are used when the sequence of items is important. Each item is numbered:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Output :

  1. First item
  2. Second item
  3. Third item

Unordered Lists

Unordered lists are used when the sequence of items is not important. Each item is bulleted:

<ul>
    <li>Red</li>
    <li>Green</li>
    <li>Blue</li>
</ul>

Output :

  • Red
  • Green
  • Blue

Definition Lists

Definition lists are used for defining terms and their corresponding definitions:

<dl>
    <dt>HTML</dt>
    <dd>Hypertext Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

Nested Lists

Lists can be nested within other lists to create a hierarchical structure:

<ul>
    <li>Main item
        <ul>
            <li>Sub-item 1</li>
            <li>Sub-item 2</li>
        </ul>
    </li>
    <li>Another main item</li>
</ul>

Output :

  • Main item
    • Sub-item 1
    • Sub-item 2
  • Another main item

Conclusion

Lists and tables are essential elements for creating well-formatted and visually appealing web content.
