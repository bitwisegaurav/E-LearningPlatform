HTML Document Structure

HTML documents follow a specific structure that includes various elements to organize and present content. Understanding the document structure is crucial for creating well-formatted and readable web pages.

Document Structure:

An HTML document typically consists of the following main elements:

  1. <!DOCTYPE html>: Declares the document type and version of HTML being used.
  2. <html>: The root element of an HTML document.
  3. <head>: Contains metadata about the document, such as the title and linked stylesheets.
  4. <meta charset="UTF-8">: Specifies the character set for the document.
  5. <title>: Sets the title of the webpage, which appears in the browser tab or window.
  6. <body>: Contains the content of the webpage, such as text, images, and other elements.

HTML Elements within <body>:

The <body> element encapsulates the content of the webpage. Within the body, various HTML elements are used to structure and present information. For example:

<body>
    <h1>Main Heading</h1>
    <p>This is a paragraph of text.</p>
    <img src="image.jpg" alt="Description of the image">
</body>

The code above includes a main heading (<h1>), a paragraph (<p>), and an image (<img>) within the body of the HTML document.

Example:

Let's consider a complete example of an HTML document:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="styles.css">
    <title>Sample HTML Document</title>
</head>
<body>
    <h1>Welcome to Our Website</h1>
    <p>Explore our services and products.</p>
    <img src="banner.jpg" alt="Website Banner">
</body>
</html>

This example demonstrates a complete HTML document with the specified structure.
