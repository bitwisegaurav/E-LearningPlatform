HTML Tables

Tables are used to organize and display data in a structured format. HTML provides a set of elements to create tables, including rows, columns, headers, and cells. Understanding how to build and style tables is essential for presenting information in a clear and organized manner.

Basic Table Structure:

The basic structure of an HTML table involves the use of the <table>, <tr> (table row), <th> (table header), and <td> (table data) elements:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
    <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
    </tr>
</table>

Output :

Header 1 Header 2
Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 Row 2, Cell 2

Table Headers and Data:

Use <th> for header cells and <td> for data cells:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
        <td>USA</td>
    </tr>
    <tr>
        <td>Alice</td>
        <td>30</td>
        <td>Canada</td>
    </tr>
</table>

Output :

Name Age Country
John 25 USA
Alice 30 Canada

Table Captions:

Add a caption to describe the purpose of the table:

<table>
    <caption>Employee Information</caption>
    <tr>
        <th>Name</th>
        <th>Department</th>
    </tr>
    <tr>
        <td>Mark</td>
        <td>HR</td>
    </tr>
    <tr>
        <td>Emily</td>
        <td>IT</td>
    </tr>
</table>

Output :

Employee Information
Name Department
Mark HR
Emily IT

Table Spanning:

Use the colspan and rowspan attributes to make a cell span multiple columns or rows:

<table>
    <tr>
        <th colspan="2">Employee Information</th>
    </tr>
    <tr>
        <td>Name</td>
        <td>Department</td>
    </tr>
    <tr>
        <td rowspan="2">Mark</td>
        <td>HR</td>
    </tr>
    <tr>
        <td>IT</td>
    </tr>
</table>

Output :

Employee Information
Name Department
Mark HR
IT

Conclusion:

The <table> tag is a fundamental element in HTML for creating structured data. It allows us to display information in rows and columns, making it easier to organize and present data. By using additional HTML elements such as <tr>, <th>, and <td>, we can create tables with headers, data cells, and captions. CSS can be applied to tables to enhance their appearance and improve readability. With the knowledge gained in this tutorial, you are now equipped to create and style tables in HTML.
