Overview of Graphics Systems-I

Introduction

Graphics systems are essential for creating and displaying visual content using computers. This chapter provides an overview of graphics systems, covering basic terms and components of various display technologies.

Basic Terms

Characteristics of Video Display

A video display device is characterized by its resolution, refresh rate, color depth, and aspect ratio.

Pixel

A pixel is the smallest unit of display on a screen, representing a single point in a graphical image.

DPI

DPI (Dots Per Inch) refers to the density of pixels on a display screen, indicating its resolution.

Monitor

CRT

A Cathode Ray Tube (CRT) is a vacuum tube that produces images when an electron beam strikes its phosphorescent surface.

Components

The components of a CRT include an electron gun, deflection coils, phosphor-coated screen, and glass envelope.

Working

The electron gun emits electron beams, which are deflected by the coils to scan across the screen, illuminating phosphor dots to create images.

Properties

CRT monitors offer high contrast, color accuracy, and response time, but they are bulky and consume more power compared to modern display technologies.

Color Monitors

Color monitors use different techniques to display colors, such as beam penetration and shadow mask.

Beam Penetration

In beam penetration, electron beams penetrate through layers of phosphor to produce different colors.

Shadow Mask

A shadow mask is a perforated metal sheet placed behind the phosphor screen to ensure that each electron beam hits the correct phosphor dot to produce colors.

DVST

Direct View Storage Tube (DVST) is a display device that stores an entire image in a memory tube, allowing fast and random access to any part of the screen.

Flat Panel Display

Flat panel displays are thin, lightweight screens that use various technologies to produce images.

Emissive Display

Emissive display technologies include Plasma Panel, Thin Film Electroluminescent, LED, and LCD.

Plasma Panel

Plasma panels use ionized gases to produce images with high brightness and color accuracy.

Thin Film Electroluminescent

Thin Film Electroluminescent displays use a thin film of phosphor sandwiched between two electrodes to emit light when an electric field is applied.

LED

LED (Light Emitting Diode) displays use an array of LEDs to produce images with high brightness and energy efficiency.

LCD

LCD (Liquid Crystal Display) uses liquid crystals to modulate light and produce images with low power consumption and high resolution.

Display Techniques

Raster Scan Display

Raster scan display generates images by scanning electron beams across the screen in a series of horizontal lines.

Components

Components include electron gun, deflection coils, phosphor-coated screen, and frame buffer.

Working

The electron beam scans across each line of the screen, illuminating phosphor dots to create images.

Random Scan Display

Random scan display directly draws images on the screen using a pen or stylus, allowing for quick and precise rendering of graphics.

Working

The pen or stylus moves across the screen to draw lines and shapes directly on the display surface.

Difference Between Raster and Random Scan System

Raster scan displays draw images using a fixed pattern of scanning lines, while random scan displays draw images directly without following a predefined pattern.

Calculate Frame Buffer Size

Frame buffer size is calculated based on the resolution, color depth, and refresh rate of the display.

Monochrome Monitor

For a monochrome monitor, the frame buffer size can be calculated using the formula:

Frame Buffer Size = Resolution (width) x Resolution (height) x Color Depth (bits per pixel)

Color Monitor

For a color monitor, the frame buffer size calculation may vary based on the display technology and color depth.
