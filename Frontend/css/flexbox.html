Flexbox in CSS

Flexbox (Flexible Box Layout) is a layout model that allows you to design complex layouts more efficiently. It provides an easier way to distribute space and align items in a container, even when their sizes are unknown or dynamic.

1. Flex Container and Flex Items:

To use Flexbox, you need to define a flex container and its child elements become flex items. Apply `display: flex;` to the container:

.container {
    display: flex;
}

2. Main Axis and Cross Axis:

Flexbox works along two axes: the main axis and the cross axis. The main axis is defined by the `flex-direction` property, and the cross axis is perpendicular to the main axis.

.container {
    display: flex;
    flex-direction: row; /* or column for vertical main axis */
}

3. Justify Content and Align Items:

Flexbox provides properties like `justify-content` to align items along the main axis and `align-items` to align items along the cross axis.

.container {
    display: flex;
    justify-content: space-between; /* Align items with space between them */
    align-items: center; /* Center items along the cross axis */
}

4. Flex Items Flexibility:

Each flex item can grow or shrink to accommodate available space. The `flex` property sets the flex grow, flex shrink, and flex basis values.

.item {
    flex: 1 0 auto; /* Flex grow: 1, Flex shrink: 0, Flex basis: auto */
}

5. Flexbox Responsive Layout:

Flexbox is excellent for creating responsive layouts. Use media queries to adjust flex properties based on screen size:

@media screen and (max-width: 600px) {
    .container {
        flex-direction: column; /* Switch to a column layout on smaller screens */
    }
}

Flexbox simplifies complex layouts, making it an essential tool for modern web development. It is particularly useful for creating navigation bars, grids, and flexible content areas.
