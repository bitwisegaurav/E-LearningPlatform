Grid Layout in CSS

Grid Layout is a powerful CSS layout system that allows you to create two-dimensional layouts with rows and columns. It provides precise control over the placement and sizing of elements within a container.

1. Creating a Grid Container:

To use Grid Layout, define a grid container using the `display: grid;` property on the parent element:

.container {
    display: grid;
}

2. Grid Template Columns and Rows:

Use the `grid-template-columns` and `grid-template-rows` properties to define the size of columns and rows in the grid:

.container {
    display: grid;
    grid-template-columns: 100px 200px 100px;
    grid-template-rows: 50px 100px;
}

3. Placing Items in the Grid:

Grid items can be placed using the `grid-column` and `grid-row` properties. You can also use the `grid-area` property for more flexibility:

.item {
    grid-column: 2 / 4; /* Place the item in columns 2 to 4 */
    grid-row: 1 / 3; /* Place the item in rows 1 to 3 */
}

4. Grid Gap:

The `grid-gap` property sets the gap between columns and rows in the grid:

.container {
    display: grid;
    grid-template-columns: 100px 200px 100px;
    grid-template-rows: 50px 100px;
    grid-gap: 10px;
}

5. Grid Auto Layout:

Grid Layout supports automatic sizing of columns and rows using `auto` and `fr` (fractional units):

.container {
    display: grid;
    grid-template-columns: auto 1fr 2fr;
}

Grid Layout is a versatile tool for creating complex layouts, including responsive designs with ease. It provides a convenient way to structure content in both rows and columns.
