Transitions and Animations in CSS

Transitions and animations bring life to web interfaces, providing smooth and engaging visual effects. CSS offers properties to control the transition between states and create dynamic animations.

1. CSS Transitions:

Transitions enable smooth changes between property values. Define the transition with the `transition` property:

.button {
    background-color: #3498db;
    color: #ffffff;
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: #2980b9;
}

2. Transition Properties:

Control the timing function, duration, delay, and property being transitioned:

.box {
    width: 100px;
    height: 100px;
    background-color: #2ecc71;
    transition: width 1s ease-in-out 0.5s, height 1s ease-in-out 0.5s;
}

.box:hover {
    width: 150px;
    height: 150px;
}

3. CSS Animations:

Animations offer more complex and dynamic effects. Define keyframes and apply them with the `animation` property:

@keyframes slide-in {
    from {
        transform: translateX(-100%);
    }
    to {
        transform: translateX(0);
    }
}

.slide {
    animation: slide-in 1s ease-in-out;
}

4. Animation Properties:

Adjust animation properties like duration, timing function, delay, and iteration count:

.rotate {
    animation: rotate 2s linear infinite;
}

@keyframes rotate {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(360deg);
    }
}

Transitions and animations enhance the user experience by adding visual interest and feedback to interactions. Use them wisely to create a more engaging and dynamic web presence.
