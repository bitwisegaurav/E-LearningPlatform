Typography and Text Styling in CSS

CSS provides a variety of properties to control the appearance of text on a web page. Proper typography and styling enhance the readability and visual appeal of content.

1. Font Properties:

Use the `font-family` property to define the font for an element. You can specify multiple fonts as fallbacks:

body {
    font-family: 'Arial', sans-serif;
    /* Fallback to sans-serif if Arial is not available */
}

h2 {
    font-family: 'Georgia', serif;
}

2. Font Size and Line Height:

Adjust the size of text using the `font-size` property. Line height can be controlled with the `line-height` property:

p {
    font-size: 16px;
    line-height: 1.5;
}

3. Font Weight and Style:

Change the thickness of characters using the `font-weight` property. Set the style to italic with `font-style`:

h3 {
    font-weight: bold;
}

em {
    font-style: italic;
}

4. Text Color and Alignment:

Modify the color of text with the `color` property. Adjust the alignment using `text-align`:

span {
    color: #007acc;
}

h4 {
    text-align: center;
}

5. Text Decoration:

Add decorations to text, such as underlines or overlines, using the `text-decoration` property:

a {
    text-decoration: underline;
}

del {
    text-decoration: line-through;
}

Mastering typography and text styling in CSS allows you to create visually appealing and well-designed web pages. Consistent styling improves the overall user experience.
