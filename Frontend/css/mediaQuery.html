Media Queries in CSS

Media queries in CSS allow you to apply different styles for different devices and screen sizes. They are a crucial part of creating responsive and adaptive web designs.

1. Basic Structure:

Define a media query using the `@media` rule. Specify the conditions under which the styles should apply:

/* Standard styles for all devices */

@media screen and (min-width: 600px) {
    /* Styles for devices with a minimum width of 600px */
}

@media (orientation: landscape) {
    /* Styles for devices in landscape orientation */
}

2. Breakpoints for Responsive Design:

Commonly used breakpoints target specific device sizes. Adjust styles based on the width of the viewport:

/* Standard styles for all devices */

@media screen and (min-width: 600px) {
    /* Styles for small devices like smartphones */
}

@media screen and (min-width: 768px) {
    /* Styles for tablets */
}

@media screen and (min-width: 1024px) {
    /* Styles for desktops and larger devices */
}

3. Responsive Typography:

Modify font sizes and spacing for better readability on different devices:

body {
    font-size: 16px; /* Default font size */

    @media screen and (min-width: 600px) {
        font-size: 18px; /* Increase font size for small devices */
    }

    @media screen and (min-width: 1024px) {
        font-size: 20px; /* Further increase font size for larger devices */
    }
}

4. Conditional Loading of Assets:

Load different images or assets based on the device's characteristics:

.image {
    background-image: url('default-image.jpg');

    @media screen and (min-width: 600px) {
        background-image: url('medium-device-image.jpg');
    }

    @media screen and (min-width: 1024px) {
        background-image: url('large-device-image.jpg');
    }
}

Media queries empower you to create designs that adapt to various screen sizes and orientations, providing an optimal user experience across different devices.
