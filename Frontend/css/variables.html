CSS Variables and Custom Properties

CSS variables, also known as custom properties, allow you to define reusable values in your stylesheets. They provide a way to store and reuse values, making it easier to maintain consistency and make global changes.

1. Declaring CSS Variables:

Define a CSS variable using the `--` prefix:

:root {
    --primary-color: #3498db;
    --font-size: 16px;
}

body {
    color: var(--primary-color);
    font-size: var(--font-size);
}

2. Scope of Variables:

Variables declared in the `:root` selector have a global scope and can be accessed anywhere in your stylesheet. However, you can also declare variables within specific selectors:

.header {
    --background-color: #2ecc71;
    background-color: var(--background-color);
}

.footer {
    background-color: var(--background-color);
}

3. Dynamic Value Changes:

Update variable values dynamically using JavaScript, allowing for theme changes or user preferences:

document.documentElement.style.setProperty('--primary-color', '#ff6347');

4. Using Variables in Animations:

Utilize variables in animations to create dynamic effects with consistent values:

@keyframes color-change {
    0% {
        background-color: var(--primary-color);
    }
    100% {
        background-color: #ffffff;
    }
}

CSS variables enhance maintainability, promote consistency, and enable dynamic styling. Experiment with them to see how they can improve your stylesheet structure.
