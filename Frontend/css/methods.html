Methods to Use CSS

Cascading Style Sheets (CSS) can be applied to HTML elements in three main ways: Inline, Internal, and External. Each method has its own use cases and advantages.

1. Inline CSS:

Inline CSS is applied directly to a single HTML element using the style attribute. This method is useful for styling specific elements on the fly. 

<p style="color: red; font-size: 16px;">This is a red paragraph with a font size of 16px.</p>

2. Internal (Embedded) CSS:

Internal CSS is placed within the <style></style> tag in the head section of an HTML document. This method allows you to apply styles to multiple elements within the same HTML file. 

<head>
    <style>
        p {
            color: blue;
            font-size: 18px;
        }
    </style>
</head>

<body>
    <p>This is a blue paragraph with a font size of 18px.</p>
</body>

3. External CSS:

External CSS involves linking an external stylesheet file to the HTML document. This method is ideal for maintaining a consistent style across multiple pages. 

<head>
    <link rel="stylesheet" href="styles.css">
</head>

The content of the `styles.css` file could be similar to this:

/* styles.css */
p {
    color: green;
    font-size: 20px;
}

External CSS files provide a modular and organized way to manage styles, promoting maintainability and reusability across your website.

When to Use Each Method:

  • Use Inline CSS for quick styling of individual elements.
  • Use Internal CSS for small-scale styling within a single HTML document.
  • Use External CSS for large-scale projects or when styling multiple pages consistently.
