Basic CSS Properties and Values

CSS properties define the visual styles of HTML elements, and each property has a corresponding value that specifies how the style should be applied. Let's explore some fundamental CSS properties and their common values.

Text Properties

color

Sets the text color. You can use named colors, hexadecimal, RGB, or HSL values.

p {
    color: #3366cc; /* Hexadecimal color */
}

font-family

Defines the font for text. You can specify multiple fonts as a fallback in case the preferred font is unavailable.

body {
    font-family: 'Arial', sans-serif; /* Fallback to sans-serif if Arial is not available */
}

font-size

Sets the size of the font. You can use various units such as pixels, em, rem, percentage, etc.

h2 {
    font-size: 24px; /* Font size in pixels */
}

Box Model Properties

margin

Sets the outer margin space of an element. You can specify values for the top, right, bottom, and left margins.

div {
    margin: 10px; /* Applies 10px margin to all sides */
}

padding

Sets the inner padding space of an element. Similar to margin, you can define padding values for individual sides.

p {
    padding: 5px 10px; /* 5px top and bottom, 10px left and right */
}

border

Defines the border of an element. It includes properties like border-width, border-style, and border-color.

div {
    border: 1px solid #999; /* 1px solid border with a color of #999 */
}

Conclusion

These are just a few examples of basic CSS properties and values. As you explore CSS further, you'll encounter a wide range of properties for styling various aspects of your web page.
