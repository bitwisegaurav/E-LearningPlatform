DOM Elements Properties in JavaScript

DOM elements in JavaScript expose various properties that allow developers to access and manipulate different aspects of the elements. These properties provide information about the element's attributes, dimensions, content, and more.

Accessing Attributes

DOM elements have properties that correspond to their attributes. For example, the id, class, href, and src properties allow developers to access the corresponding attributes of elements.

// Accessing the id attribute of an element
let link1 = document.getElementById('myLink');

// Accessing the class attribute of an element
let link2 = document.getElementsByClasses('myLink');

// Accessing the href attribute of an element
let link3 = document.querySelector('[href="example.com"]');

// Accessing the src attribute of an element
let link4 = document.querySelector('[src="example.com"]');

console.log(link1, link2, link3, link4);

Element Content

Properties like textContent, innerHTML, and innerText provide access to the content of elements. These properties allow developers to retrieve and modify the text and HTML content of elements.

Difference between them is :

  • innerHTML property sets or returns the HTML content of an element.
  • innerText property sets or returns the text content of an element which is visible to the user.
  • textContent property sets or returns the text content of an element whether it is visible or not.

For example : In html code

<p>
    JavaScript 
    <span style="display: none;">, often abbreviated as JS</span>
</p>

In js code : 

let p = document.querySelector('p');
console.log(`- innerHTML = "${p.innerHTML}"`);
console.log(`- innerText = "${p.innerText}"`);
console.log(`- textContent = "${p.textContent}"`);

Result : 

- innerHTML = "JavaScript 
<span style="display: none;">, often abbreviated as JS</span>"
- innerText = "JavaScript"
- textContent = "JavaScript , often abbreviated as JS"

Element Dimensions

Properties such as clientWidth, clientHeight, offsetWidth, and offsetHeight provide information about the dimensions of elements. These properties are useful for determining the size and position of elements on the page.

Other Properties

DOM elements may expose additional properties depending on their type and attributes. For example, input elements have properties like value, checked, and disabled, while image elements have properties like naturalWidth and naturalHeight.

Example : 


let input = document.getElementById('myInput');
console.log(input.value);

Conclusion

DOM elements expose a wide range of properties that provide access to different aspects of the elements. By leveraging these properties, developers can retrieve information about element attributes, content, dimensions, and more, enabling them to create dynamic and interactive web applications.
