Operators in JavaScript

Operators in JavaScript are symbols that perform various operations on variables and values, allowing for mathematical calculations, logical comparisons, and more. Understanding the different types of operators is crucial for writing expressive and effective code.

Arithmetic Operators

Arithmetic operators perform mathematical operations on numeric values. These include:

  • Addition (+) : Adds two operands.
  • Subtraction (-) : Subtracts the right operand from the left operand.
  • Multiplication (*) : Multiplies two operands.
  • Division (/) : Divides the left operand by the right operand.
let num1 = 10;
let num2 = 5;

let sum = num1 + num2; // 10 + 5 = 15
let difference = num1 - num2; // 10 - 5 = 5
let product = num1 * num2; // 10 * 5 = 50
let quotient = num1 / num2; // 10 / 5 = 2

Comparison Operators

Comparison operators compare two values and return a boolean result. Common comparison operators include:

  • Equality (==) : Checks if two values are equal, performing type coercion if necessary.
  • Strict Equality (===) : Checks if two values are equal without type coercion.
  • Inequality (!=) : Checks if two values are not equal, performing type coercion if necessary.
  • Strict Inequality (!==) : Checks if two values are not equal without type coercion.
let isEqual = (num1 === num2); // Strict equality check
let isNotEqual = (num1 !== num2); // Strict inequality check

Logical Operators

Logical operators perform logical operations and return a boolean result. These include:

  • Logical AND (&&) : Returns true if both operands are true.
  • Logical OR (||) : Returns true if at least one operand is true.
  • Logical NOT (!) : Returns the opposite of the operand's truthiness.
let logicalAnd = (true && false); // false (AND operator)
let logicalOr = (true || false); // true (OR operator)
let logicalNot = !true; // false (NOT operator)

Assignment Operators

Assignment operators assign values to variables. The basic assignment operator is:

  • Assignment (=) : Assigns the value on the right to the variable on the left.
let x = 5;
x += 3; // Equivalent to x = x + 3

5. Unary Operators

Unary operators operate on a single operand. Common unary operators include:

  • Increment (++) : Increases the value of a variable by 1.
  • Decrement (--) : Decreases the value of a variable by 1.
let counter = 10;
counter++; // Increment by 1
counter--; // Decrement by 1

Conclusion

Operators are fundamental to JavaScript programming, enabling a wide range of operations on variables and values. Whether performing arithmetic calculations, logical comparisons, or assignments, using operators effectively enhances the expressiveness and functionality of your code.
