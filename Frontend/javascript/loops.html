Loops in JavaScript

Loops in JavaScript are used to execute a block of code repeatedly as long as a specified condition is truhey provide a convenient way to iterate over arrays, manipulate data, and perform repetitive tasks in a more efficient manner.

For Loop

The for loop is a common choice for iterating a specific number of timet consists of an initialization, a condition, and an increment or decrement statement.

for (let i = 0; i < 5; i++) {
    console.log("Iteration: " + i);
}

While Loop

The while loop repeats a block of code as long as a specified condition is trut's suitable when the number of iterations is not known beforehand.

let counter = 0;

while (counter < 3) {
    console.log("Counting: " + counter);
    counter++;
}

Do-while Loop

The do-while loop is similar to the while loop, but it guarantees that the block of code is executed at least once, even if the condition is initially false.

let num = 5;

do {
    console.log("Number is: " + num);
    num--;
} while (num > 0);

For...in Loop

The for...in loop iterates over the enumerable properties of an object, providing a convenient way to loop through object properties.

const person = { name: "John", age: 30, job: "Developer" };

for (let key in person) {
    console.log(key + ": " + person[key]);
}

For...of Loop

The for...of loop is used to iterate over iterable objects such as arrays, strings, and mort provides a simpler syntax compared to the traditional for loop.

const colors = ["red", "green", "blue"];

for (let color of colors) {
    console.log("Color: " + color);
}

Conclusion

Loops are powerful constructs in JavaScript, enabling developers to efficiently perform repetitive tasks and iterate over data structurehether using the classic for loop or the modern for...of loop, understanding these constructs is key to writing efficient and maintainable code.
