Getter and Setter in JavaScript

Getter and setter methods are special functions that are used to get and set the values of an object's properties, respectively. They provide a way to control access to object properties and add additional behavior when getting and setting values.

1. Getter Methods

Getter methods are used to retrieve the value of a property. They are defined using the get keyword followed by the method name. When the property is accessed, the getter method is automatically invoked.

2. Example:

// Example of getter method
class Circle {
    constructor(radius) {
        this.radius = radius;
    }

    get area() {
        return Math.PI * this.radius * this.radius;
    }
}

const circle = new Circle(5);
console.log(circle.area); // Output: 78.53981633974483

3. Setter Methods

Setter methods are used to set the value of a property. They are defined using the set keyword followed by the method name. When the property is assigned a new value, the setter method is automatically invoked.

4. Example:

// Example of setter method
class Square {
    constructor(side) {
        this.side = side;
    }

    get area() {
        return this.side * this.side;
    }

    set area(value) {
        this.side = Math.sqrt(value);
    }
}

const square = new Square(4);
console.log(square.area); // Output: 16
square.area = 25;
console.log(square.side); // Output: 5

Conclusion

Getter and setter methods provide a way to control access to object properties and add additional behavior when getting and setting values. By leveraging getter and setter methods, developers can create more flexible and robust object-oriented code in JavaScript.
