Conditional Statements in JavaScript

Conditional statements in JavaScript allow you to control the flow of your code based on specified conditions. These statements make decisions and execute different code blocks depending on whether a condition evaluates to true or false.

If Statement

The if statement is the most basic conditional statement. It executes a block of code if the specified condition is true.

let num = 10;

if (num > 0) {
    console.log("The number is positive.");
}

Else Statement

The else statement is used in conjunction with if to execute a different block of code if the initial condition is false.

let num = -5;

if (num > 0) {
    console.log("The number is positive.");
} else {
    console.log("The number is non-positive.");
}

Else if Statement

The else if statement allows you to specify multiple conditions to check in sequence. The code block associated with the first true condition is executed.

let num = 0;

if (num > 0) {
    console.log("The number is positive.");
} else if (num < 0) {
    console.log("The number is negative.");
} else {
    console.log("The number is zero.");
}

Switch Statement

The switch statement provides a way to handle multiple conditions in a more concise manner than a series of else if statements.

let day = "Monday";

switch (day) {
    case "Monday":
        console.log("It's the start of the week.");
        break;
    case "Friday":
        console.log("It's almost the weekend.");
        break;
    default:
        console.log("It's a regular day.");
}

Conclusion

Conditional statements are essential for building dynamic and responsive programs. Whether making simple decisions with if statements or handling multiple conditions with switch, mastering these constructs is fundamental to effective JavaScript programming.
