DOM in JavaScript

The DOM (Document Object Model) is an API that represents and interacts with any HTML or XML-based markup language document. The DOM is a document model loaded in the browser and representing the document as a node tree, or DOM tree, where each node represents part of the document (e.g. an element, text string, or comment).

It allows code running in a browser to access and interact with every node in the document. Nodes can be created, moved, and changed.

API (Application Programming Interface) allows different software programs to talk to each other and share information easily. It's like a waiter taking orders between two people.

DOM Tree Structure

The DOM tree consists of nodes representing various elements of an HTML document, such as elements, attributes, and text. Each node in the tree can be accessed, modified, or removed using JavaScript, enabling developers to create dynamic and interactive web applications.

One example this is : 

<html>
    <head>
        <title>My title</title>
    </head>
    <body>
        <a href="example.com">My link</a>
        <h1>My header</h1>
    </body>
</html>

Accessing DOM Elements

JavaScript provides several methods for accessing DOM elements. These methods allow developers to select and manipulate specific elements on a web page based on their IDs, classes, or other attributes.

By Id

The getElementById() method returns the element with the specified ID. This method can be used to select a single element from the DOM based on its unique ID attribute.

For example, the following HTML markup contains a single element with an ID of my-element : 

<div id="my-element">This is my element</div>

The following JavaScript code selects the element with the ID of my-element and displays its content in console : 

const myElement = document.getElementById('my-element');
console.log(myElement);

It returns the element having this Id.

By Class Name

You can also access elements by their class name using the getElementsByClassName() method. This method returns a collection of elements that have the specified class name.

In the DOM, the class attribute is referred to as className.

For example, the following HTML markup contains two elements with the class name of my-class : 

<div class="my-class">This is my first element</div>
<div class="my-class">This is my second element</div>

The following JavaScript code selects all elements with the class name of my-class and displays their content in console : 

const myElements = document.getElementsByClassName('my-class');
console.log(myElements);
console.log(myElements[0]);
// Returns NodeList(2) [div, div]

The getElementsByClassName() method returns a HTMLCollection object, which is a collection of nodes. You can access individual elements in the collection using array-like syntax, as shown in the example above.

But you cannot apply array methods(map, forEach etc. or any other loop) directly to a HTMLCollection. You can use Array.from() to convert HTMLCollection to an array.

By Tag Name

The getElementsByTagName() method returns a collection of elements with the specified tag name. This method can be used to select all elements of a particular type, such as div, p, or span.

For example, the following HTML markup contains two elements with the tag name of p : 

<p>This is my first paragraph</p>

The following JavaScript code selects all elements with the tag name of p and displays their content in console : 

const myParagraphs = document.getElementsByTagName('p');
console.log(myParagraphs); 
// Returns HTMLCollection(1) [p]

The getElementsByTagName() method also returns an HTMLCollection object.

By Query Selector

The querySelector() method returns the first element that matches the specified CSS selector and the querySelectorAll method returns all the elements that mathes CSS selector. These methods can be used to select elements based on their IDs, classes, attributes, or other CSS selectors.

For example, the following HTML markup contains a mulitple elements with class of my-elements : 

<div class="my-elements">This is my element</div>
<div class="my-elements">This is my element</div>

The following JavaScript code selects the element with the ID of my-element and displays its content in console : 

const myElement = document.querySelector('.my-elements');
const myElements = document.querySelectorAll('.my-elements');
console.log(myElement); // Returns the first element
console.log(myElements); // Return the NodeList(2) [div, div]

To apply high order loops on a NodeList, it needs to be converted into an Array first.

Modifying DOM Elements

DOM elements can be modified dynamically using JavaScript. This includes changing element attributes, adding or removing classes, modifying element content, and manipulating element styles. These modifications can be triggered by user interactions, events, or programmatic logic.

Changing Element Attributes

Changing element attributes allows you to modify the behavior or appearance of an element. To do this, you can access an element using the getElementById(), querySelector(), or other methods, and then modify its attributes using the setAttribute() method.

For instance, if you want to change the href attribute of a link element: 

let linkElement = document.querySelector("a");
linkElement.setAttribute("href", "https://example.com");

Adding or Removing Classes

Adding or removing classes can help you dynamically apply or remove styles. To add a class to an element, you can use the classList.add() method, and to remove a class, you can use the classList.remove() method.

For example, if you want to add or remove a class called active to a button element: 

let buttonElement = document.querySelector("button");
buttonElement.classList.add("active"); // Add the "active" class
buttonElement.classList.remove("active"); // Remove the "active" class

Modifying Element Content

Modifying element content can help you update the text or HTML within an element. To change the content, you can use the innerHTML, innerText or textContent properties.

Difference between innerHTML, innerText and textContent properties is that :
- innerHTML property sets or returns the HTML content of an element.
- innerText property sets or returns the text content of an element which is visible to the user.
- textContent property sets or returns the text content of an element whether it is visible or not.

For example, if you want to change the content of a paragraph element: 

let paragraphElement = document.querySelector("p");
paragraphElement.innerHTML = "This is the new content.";
paragraphElement.textContent = "This is the new content.";

Manipulating Element Styles

Manipulating element styles can help you dynamically update the appearance of an element. To change the style, you can use the style property and access the CSS property you want to modify, such as color or background-color.

For example, if you want to change the background color of a div element: 

let divElement = document.querySelector("div");
divElement.style.backgroundColor = "red";

These modifications can be triggered by user interactions, events, or programmatic logic.

Handling Events

JavaScript enables developers to attach event handlers to DOM elements to respond to user interactions, such as clicks, mouse movements, keyboard inputs, and form submissions. Event handling allows developers to create interactive and responsive web applications by defining how the application should respond to various user actions.

Conclusion

The DOM is a fundamental concept in web development, enabling JavaScript to interact with and manipulate web documents dynamically. Understanding how to access, modify, and handle events on DOM elements is essential for building modern web applications with rich user interfaces and interactive functionality.
