Some Important Objects

JavaScript leverages built-in objects to encapsulate functionalities related to mathematical operations, date handling, and numerical values. Let's explore some of these objects and their methods :

Number Object

The Number object provides methods for numerical operations and conversions:

const numberOperations = {
    parseStringToNumber: (stringValue) => Number(stringValue),
    formatWithDecimals: (number, decimalPlaces) => number.toFixed(decimalPlaces),
    checkInteger: (value) => Number.isInteger(value)
};

let parsedNumber = numberOperations.parseStringToNumber("42");
let fixedDecimal = numberOperations.formatWithDecimals(parsedNumber, 2);
let isInteger = numberOperations.checkInteger(7.5);

Math Object

The Math object encapsulates mathematical constants and functions:

const mathOperations = {
    generateRandomNumber: () => Math.random(),
    raiseToPower: (base, exponent) => Math.pow(base, exponent),
    roundNumber: (number) => Math.round(number)
};

let randomNumber = mathOperations.generateRandomNumber();
let squaredNumber = mathOperations.raiseToPower(4, 2);
let roundedNumber = mathOperations.roundNumber(5.67);

Math Constants

The Math object includes constants for common mathematical calculations:

const mathConstants = {
    calculateCircumference: (radius) => 2 * Math.PI * radius,
    calculateExponentialValue: (exponent) => Math.exp(exponent)
};

let circumference = mathConstants.calculateCircumference(5);
let exponentialValue = mathConstants.calculateExponentialValue(2);

Date Object

The Date object facilitates date and time manipulation:

const dateOperations = {
getCurrentDate: () => new Date(),
createDate: (dateString) => new Date(dateString),
getCurrentHour: (date) => date.getHours(),
getDayOfWeek: (date) => date.getDay()
};

let currentDate = dateOperations.getCurrentDate();
let specificDate = dateOperations.createDate("2024-03-02");
let hours = dateOperations.getCurrentHour(currentDate);
let dayOfWeek = dateOperations.getDayOfWeek(currentDate);

Conclusion

Encapsulating functionalities within objects like Math, Date, and Number promotes cleaner code organization and improves readability. These objects play a crucial role in various web development scenarios, offering powerful tools for mathematical operations, date manipulation, and numerical value handling.
