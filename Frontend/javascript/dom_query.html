matches, closest, and contains in JavaScript

In JavaScript, the matches(), closest(), and contains() methods provide convenient ways to work with DOM elements based on their relationships with other elements.

matches()

The matches() method is used to check if an element matches a specified CSS selector. It returns true if the element matches the selector, and false otherwise. This method is useful for conditionally applying styles or behaviors to elements based on their attributes or classes.

Example : 


let element = document.getElementById('myElement');
if (element.matches('.highlight')) {
    console.log('Element matches .highlight selector');
}

closest()

The closest() method is used to find the closest ancestor of an element that matches a specified CSS selector. It traverses up the DOM tree from the current element, testing each ancestor until a match is found or the top of the DOM tree is reached. This method is useful for finding parent elements or navigating the DOM hierarchy.

Example : 


let parent = element.closest('.parent');
if (parent) {
    console.log('Closest parent with class .parent found');
}

contains()

The contains() method is used to check if one DOM element is a descendant of another element. It returns true if the specified element is a descendant of the calling element, and false otherwise. This method is useful for checking if an element contains specific content or child elements.

Example : 


let container = document.getElementById('container');
let child = document.getElementById('child');
if (container.contains(child)) {
    console.log('container contains child element');
}

Conclusion

The matches(), closest(), and contains() methods provide convenient ways to work with DOM elements based on their relationships with other elements. By leveraging these methods, developers can easily query and manipulate the DOM to create dynamic and interactive web applications.
