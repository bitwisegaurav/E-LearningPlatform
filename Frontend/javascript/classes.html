Classes in JavaScript

Classes are a fundamental concept in object-oriented programming (OOP) that represent blueprints for creating objects with shared properties and methods. JavaScript introduced class syntax in ECMAScript 2015 (ES6) to provide a more familiar and structured way of defining objects and their behavior.

1. Class Declaration

In JavaScript, classes are declared using the class keyword followed by the class name and class body, which contains the constructor method and other class methods.

2. Example:

// Example of class declaration
class Rectangle {
    constructor(width, height) {
        this.width = width;
        this.height = height;
    }

    area() {
        return this.width * this.height;
    }
}

const rectangle = new Rectangle(10, 5);
console.log(rectangle.area()); // Output: 50

3. Class Inheritance

Classes support inheritance, allowing subclasses (child classes) to inherit properties and methods from parent classes. This enables code reuse and promotes a hierarchical structure of classes.

4. Example:

// Example of class inheritance
class Square extends Rectangle {
    constructor(side) {
        super(side, side);
    }
}

const square = new Square(5);
console.log(square.area()); // Output: 25

5. Static Methods

Static methods are methods that belong to the class itself rather than instances of the class. They are defined using the static keyword and can be called directly on the class.

Conclusion

Classes are a powerful feature in JavaScript for creating blueprints for objects with shared properties and methods. By leveraging class syntax and inheritance, developers can create more structured, maintainable, and reusable code.
