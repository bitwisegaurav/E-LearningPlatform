Map, Filter, and Reduce in JavaScript

Map, Filter, and Reduce are powerful array methods in JavaScript that facilitate concise and efficient operations on arrayThey allow for transforming, filtering, and aggregating data, providing a functional programming approach to array manipulation.

Map

The map method creates a new array by applying a function to each element of the original array.

let numbers = [1, 2, 3, 4, 5];

let squaredNumbers = numbers.map(function(number) {
    return number ** 2;
});

// Result: [1, 4, 9, 16, 25]

Filter

The filter method creates a new array with elements that satisfy a specified condition.

let evenNumbers = numbers.filter(function(number) {
    return number % 2 === 0;
});

// Result: [2, 4]

Reduce

The reduce method reduces an array to a single value by applying a function to each element and accumulating the result.

let sum = numbers.reduce(function(accumulator, currentValue) {
    return accumulator + currentValue;
}, 0);

// Result: 15

Chaining Methods

These methods can be chained together for more complex operations in a single line.

let result = numbers
    .map(function(number) { return number * 2; })
    .filter(function(number) { return number > 5; })
    .reduce(function(accumulator, currentValue) { return accumulator + currentValue; }, 0);

// Result: 18

Arrow Functions with Map, Filter, and Reduce

Arrow functions provide a concise syntax for these methods.

let squaredNumbers = numbers.map(number => number ** 2);
let evenNumbers = numbers.filter(number => number % 2 === 0);
let sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);

Conclusion

Map, Filter, and Reduce are integral parts of modern JavaScript programming, offering elegant solutions for manipulating and transforming arrayWhether you're mapping values, filtering elements, or reducing to a single result, these methods contribute to cleaner and more maintainable code.
