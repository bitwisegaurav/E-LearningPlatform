Inline, Internal, and External

JavaScript can be included in HTML documents in various ways. Understanding these methods is essential for organizing and maintaining your code efficiently.

Inline JavaScript

Inline JavaScript is used within the tags.

For example : 

<html>
<body>
    <h2>This is an inline script</h2>
    <button onclick="function() { getBgRed(); }" >Click me</button>
</body>
</html>

Internal JavaScript

Internal JavaScript is placed within the <script> tags in the head or body of the HTML document:

<html>
<head>
    <title>Internal JavaScript</title>
    <script>
        // Your JavaScript code here
        function greet() {
            alert("Hello, Internal!");
        }
    </script>
</head>
<body>
    <h2>Call the greet function</h2>
    <button onclick="greet()">Click me</button>
</body>
</html>

External JavaScript

External JavaScript is stored in separate files with a .js extension. It can be linked to an HTML document using the <script> tag with the src attribute:

Linking the external file in HTML:

<html>
<head>
    <title>External JavaScript</title>
    <script src="external.js"></script>
</head>
<body>
    <h2>This content is modified by external JavaScript</h2>
</body>
</html>

In external.js : 

document.getElementById("demo").innerHTML = "Hello, External!";

Choose the method that suits your project's structure and requirements.
