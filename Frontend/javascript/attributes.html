DOM Elements Attributes in JavaScript

In JavaScript, DOM elements have various attributes that define their characteristics and behavior. These attributes can be accessed, modified, or removed dynamically using JavaScript, allowing developers to create interactive and dynamic web pages.

Accessing Attributes

DOM elements have properties that correspond to their attributes. For example, the id, class, href, and src properties allow developers to access the corresponding attributes of elements.

Example : 


let link = document.getElementById('myLink');

// Accessing the id attribute
let id = link.id; 

// Accessing the class attribute
let classes = link.classList; 

// Accessing the href attribute
let href = link.href; 

// Accessing the src attribute
let src = link.src; 

// Access any attribute
let attribute = link.getAttribute('attributeName');

Modifying Attributes

Attributes of DOM elements can be modified using their properties or by using the setAttribute() method. This allows developers to dynamically change the attributes of elements based on user interactions or other events.

Example : 


let link = document.getElementById('myLink');
link.setAttribute("attributeName", value);
link.setAttribute("href", "bitwisegaurav.github.io");

Removing Attributes

Attributes of DOM elements can be removed using the removeAttribute() method. This allows developers to remove attributes from elements when they are no longer needed or to reset the state of elements.

Example : 


let paragraph = document.getElementById('myParagraph');
paragraph.removeAttribute('class');

Conclusion

DOM elements in JavaScript have various attributes that define their characteristics and behavior. By accessing, modifying, and removing these attributes dynamically, developers can create interactive and dynamic web pages that respond to user interactions and events.
