Relations in Discrete Mathematics

What is a Relation?

A relation from A to B is a subset of A x B.

or

We can say R ⊂ A x B.

Domain

The domain of a relation is the set of elements from which the relation is defined.

Range

The range of a relation is the set of elements that are the output or result of the relation.

In other words, it is the set of all values that the relation maps to.

Co-Domain

The co-domain of a relation is the set of all possible values that the output or result of the relation can take. It represents the entire set of potential outputs, regardless of whether they are actually mapped to by the relation.

Types of Relations

  • Reflexive Relation : A relation R on a set A is said to be reflexive if every element is related to itself. It means R is reflexive if (a, a) ∈ R for every a ∈ A.

    For e.g. R = {(1,1),(2,2)}.

  • Symmetric Relation : A relation R on a set A is said to be symmetric if for every (a, b) in R, (b, a) is also in R. It means R is symmetric if for all (a, b) ∈ R implies that (b, a) ∈ R.

    For e.g. R = {(1,2),(2,1)}.

  • Transitive Relation : A relation R on a set A is said to be transitive if for every (a, b) and (b, c) in R, (a, c) is also in R. It means R is transitive if (a, b) ∈ R and (b, c) ∈ R then (a, c) ∈ R.

    For e.g. R = {(1,2),(2,3),(1,3)}.

  • Anti-Symmetric Relation : A relation R on a set A is said to be anti-symmetric if for every (a, b) in R, where a ≠ b, (b, a) is not in R. In other words, if (a, b) is in R, then (b, a) cannot be in R when a and b are distinct elements.
  • Inverse Relation : Given a relation R on a set A, the inverse relation of R, denoted as R⁻¹, is a relation that reverses the order of the elements in R. If (a, b) is in R, then (b, a) is in R⁻¹.

    For e.g. R = {(1,2),(2,3),(1,3)} then R -1 = {(2,1),(3,2),(3,1)}.

  • Void Relation : A relation R on a set A is said to be void if it does not contain any elements. In other words, φ or null set.
  • Universal Relation : A relation R on a set A is called a universal relation if it contains all possible pairs of elements from A. In other words, R is a universal set if R be the set A x A where A is any set.
  • Compatible Relation : A relation which is reflexive and symmetric.
  • Partial Order Relation : A relation which is reflexive and transitive.

Adjacency Matrix

An adjacency matrix is a square matrix used to represent a finite graph. The elements of the matrix indicate whether pairs of vertices are adjacent or not in the graph. In the special case of a finite simple graph, the adjacency matrix is a (0,1)-matrix with zeros on its diagonal.

Closures

In mathematics, the closure of a subset S in a topological space consists of all points in S along with the limit points of S. In other words, the closure of S contains all points that are arbitrarily "close" to S. A point which is in the closure of S is a point of closure of S.

Hasse Diagram / Ordering Diagram

A Hasse diagram is a graphical representation of the partial order relation on a set. It is a drawing of the transitive reduction of the relation, meaning it shows the minimal set of direct connections that imply all the connections in the relation via transitivity.
