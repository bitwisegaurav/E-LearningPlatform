Recursion and Recurrence Relations

Sequence

A sequence is a set of numbers that are related to each other by a rule.

For example 1, 2, 3, 4, ... etc. Here an = n.

Recursion

Recursion is a method where the solution to a problem depends on solutions to smaller instances of the same problem.

or

A technique of defining a function, a set or an algorithm in terms of itself is a recursion.

For example 0, 1, 1, 2, 3, 5, 8, ... etc. Here an+1 = an + 1.

Recurrence Relation

A recurrence relation is an equation that recursively defines a sequence where the next term is a function of the previous terms.

or

A relation which is recursive in nature is known as recurrence relation.

A recurrence relation is also called a difference equation.

Important Terms

  • Base Case : The first term of a sequence.
  • Boundary Conditions : Given values of functions.
  • Induction Step : The step by which a sequence is built.
  • Order : The order of a recurrence relation is the difference between highest and lowest subscript.
  • Characteristic Equation : It is an equation whose roots are the values of n for which the recurrence relation holds.

    For example: an = 2an-1 + 1, the characteristic equation is r2 - 2r - 1 = 0.

  • General Solution : A general solution of a recurrence relation is a sequence of numbers such that the recurrence relation holds for all the terms of the sequence and the values of the constants in the recurrence relation are known. For example: an = 2an-1 + 1, the general solution is 1, 2, 4, 8, 1

    The general solution is also called the general term.

  • Particular Solution : A particular solution of a recurrence relation is a solution that satisfies the recurrence relation and the initial conditions. It is obtained by solving the recurrence relation with the given initial conditions.
  • Total Solution : The total solution of a recurrence relation is the sum of the general solution of the homogeneous equation and a particular solution of the non-homogeneous equation.
  • Linear Recurrence Relation : A linear recurrence relation is a recurrence relation in which every term of the sequence is a linear function of its previous terms.
  • Homogeneous Recurrence Relation : A homogeneous recurrence relation is a type of recurrence relation where each term is a constant multiple of its preceding term.

    or

    If the right-hand side of the recurrence relation is zero, it is called a homogeneous recurrence relation.

Types of Recurrence Relations

  • First Order Recurrence Relation

    A linear recurrence relation of the first order with constant coefficients is defined as:

    an = can-1 + f(n)

    where 'a' is the sequence, 'c' is a constant, and f(n) is a known function. This relation holds for n greater than or equal to 1. If f(n) is equal to zero, the relation is considered homogeneous; otherwise, it is non-homogeneous. For example: xn = 2xn-1 - 1, an = nan-1 + 1, etc.

  • Second Order Linear Homogeneous Recurrence Relation

    A second order linear homogeneous recurrence relation with constant coefficients is defined as:

    cnan + cn-1an-1 + cn-2an-2 = 0

    This relation holds for n greater than or equal to 2, where cn, cn-1, and cn-2 are real constants with cn not equal to zero.

Solving Homogeneous Recurrence Relations

There are different methods to solve recurrence relations, including generating functions.

Sk + C1Sk-1 + C2Sk-2 + ... + CnSk-n = 0

Steps to Solve it

  1. Write down the characterstic equation. 
    an + C1an-1 + C2an-2 + ... + Cn-1a1 + Cn = 0
  2. Solve equation and find its roots. Like a1,a2, ..., an.
  3. Solve the equation. The solutions depend on the roots of the characteristic equation associated with the relation.
Cases
  1. If the roots are real and distinct, we take the general solution. 
    Sk = b1a1k + b2a2k + b3a3k + ... + bnank
  2. If the roots are real and equal, we take the general solution if a1 = a2 then. 
    Sk = (b1 + b2)a1k + b3a3k + ... + bnank
  3. If the roots are complex, we take an = rn(Acos(nθ) + Bsin(nθ)) as the general solution.

Solution of Non-Homogeneous Finite Order Linear Recurrence Relation

A non-homogeneous finite order linear recurrence relation with constant coefficients is defined as:

an = an-1 + bnf(n)

where 'a' is the sequence, 'b' is a constant, and f(n) is a known function. This relation holds for n greater than or equal to 1. If f(n) is equal to zero, the relation is considered homogeneous; otherwise, it is non-homogeneous. For example: xn = 2xn-1 - 1, an = nan-1 + 1, etc.

Generating Functions

A generating function is a function that generates a sequence of numbers. It is defined as:

a0 + a1x + a2x2 +... + anxn = 0

or
