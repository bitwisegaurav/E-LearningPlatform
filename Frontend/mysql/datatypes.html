MySQL Data Types

MySQL supports a wide range of data types that can be used to store and manipulate different types of data. These data types can be categorized into several groups :

  • Numeric Data Types
  • Date and Time Data Types
  • String Data Types
  • Binary Data Types
  • Bit-Value Data Types
  • Enumerated Data Types
  • Set Data Types

Numeric Data Types

Numeric data types are used to store numeric values. MySQL supports several types of numeric data:

  • INT : A whole number between -2147483648 and 2147483647. Can be signed or unsigned.
  • SMALLINT : A whole number between -32768 and 32767. Can be signed or unsigned.
  • TINYINT : A whole number between -128 and 127. Can be signed or unsigned.
  • BIGINT : A whole number between -9223372036854775808 and 9223372036854775807. Can be signed or unsigned.
  • DECIMAL : A fixed-point number with up to 65 digits, with a specified number of digits to the right of the decimal point.
  • FLOAT : A floating-point number with up to 23 significant digits.
  • DOUBLE : A floating-point number with up to 53 significant digits.

Date and Time Data Types

Date and time data types are used to store date and time values. MySQL supports several types of date and time data:

  • DATE : A date in YYYY-MM-DD format.
  • DATETIME : A date and time in YYYY-MM-DD HH:MM:SS format.
  • TIME : A time in HH:MM:SS format.
  • TIMESTAMP : A timestamp in YYYY-MM-DD HH:MM:SS format.
  • YEAR : A year in YYYY or YY format.

String Data Types

String data types are used to store character or text values. MySQL supports several types of string data:

  • CHAR : A fixed-length string of up to 255 characters.
  • VARCHAR : A variable-length string of up to 65535 characters.
  • BLOB : A binary large object that can store up to 4GB of data.
  • TEXT : A text object that can store up to 65535 characters.
