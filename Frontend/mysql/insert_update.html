Insert and Update rows

Insert single row

Syntax :

mysql> insert into table_name (first_column_name,second_column_name,...) values (first value, second value,...);

or

mysql> insert into table_name values (first value, second value,...);

Example :

Inserting multiple rows

Syntax :

mysql> insert into table_name values (first value, second value,...) , (first value, second value,...);

Example :

Updating row

Syntax :

mysql> update table_name set col_name=value, col_name=value,... where condition;
