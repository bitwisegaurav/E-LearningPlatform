PL/SQL Data Types

PL/SQL has a variety of data types that can be used to define variables, constants, and parameters.

  • NUMBER: This data type is used to store numeric values with a precision of up to 38 digits. It includes both integers and decimal numbers.
  • VARCHAR2: This data type is used to store character strings with a maximum length of 4000 bytes.
  • DATE: This data type is used to store date and time values in the format of DD-MON-YYYY HH:MI:SS.
  • BOOLEAN: This data type is used to store Boolean values, i.e., TRUE or FALSE.
  • BINARY_INTEGER: This data type is used to store integers with a range of -2147483647 to 2147483647.
  • PLS_INTEGER: This data type is used to store integers with a range of -2147483647 to 2147483647. It is faster and takes less memory than BINARY_INTEGER.
  • CHAR: This data type is used to store fixed-length character strings with a maximum length of 2000 bytes.
  • CLOB: This data type is used to store large character strings with a maximum size of 4GB.
  • BLOB: This data type is used to store large binary data with a maximum size of 4GB.
