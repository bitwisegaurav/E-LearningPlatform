Decision Control Structures

PL/SQL provides the following decision control structures:

  • IF-THEN: The IF-THEN statement executes a sequence of statements if a condition is true.
  • IF-THEN-ELSE: The IF-THEN-ELSE statement executes one sequence of statements if a condition is true and another sequence of statements if the condition is false.
  • IF-THEN-ELSIF: The IF-THEN-ELSIF statement executes one sequence of statements if a condition is true and another sequence of statements if the condition is false, but additional conditions can be tested.

Example of IF-THEN statement:

DECLARE
	num1 NUMBER := 10;
BEGIN
	IF num1 > 5 THEN
		DBMS_OUTPUT.PUT_LINE('num1 is greater than 5');
	END IF;
END;

Example of IF-THEN-ELSE statement:

DECLARE
	num1 NUMBER := 3;
BEGIN
	IF num1 > 5 THEN
		DBMS_OUTPUT.PUT_LINE('num1 is greater than 5');
	ELSE
		DBMS_OUTPUT.PUT_LINE('num1 is less than or equal to 5');
	END IF;
END;

Example of IF-THEN-ELSIF statement:

DECLARE
	num1 NUMBER := 7;
BEGIN
	IF num1 < 5 THEN
		DBMS_OUTPUT.PUT_LINE('num1 is less than 5');
	ELSIF num1 < 10 THEN
		DBMS_OUTPUT.PUT_LINE('num1 is between 5 and 10');
	ELSE
		DBMS_OUTPUT.PUT_LINE('num1 is greater than or equal to 10');
	END IF;
END;
