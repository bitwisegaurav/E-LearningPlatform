Iterative Control Structures

Iterative control structures in PL/SQL are used to repeat a block of code until a certain condition is met. There are three types of iterative control structures in PL/SQL:

  • SIMPLE loop : It is mainly used for infinite loops.
  • FOR loop : It is used to iterate over a specified range of numbers or a collection.
  • WHILE loop : It is used to repeat a block of code until a certain condition is met.

SIMPLE loop

The syntax of a SIMPLE loop is as follows:

LOOP
    -- Statements to be executed
END LOOP;

We can also terminate loop using exit 

exit when condition;

FOR loop

The syntax of a FOR loop is as follows:

FOR loop_counter IN [REVERSE] lower_bound..upper_bound LOOP
    -- Statements to be executed
END LOOP;

Here, loop_counter is a variable used to count the iterations, lower_bound is the starting value of the loop counter, upper_bound is the ending value of the loop counter, and REVERSE is an optional keyword used to reverse the direction of the loop.

WHILE loop

The syntax of a WHILE loop is as follows:

WHILE condition LOOP
    -- Statements to be executed
END LOOP;

The condition is a Boolean expression that is evaluated at the beginning of each iteration. If the condition evaluates to TRUE, the loop continues. If the condition evaluates to FALSE, the loop terminates.
